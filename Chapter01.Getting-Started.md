# Graphics System (图形系统)

两大图形系统
- Raster Graphics 光栅图形
- Vector Graphics 矢量图形

## Raster Graphics (光栅图形)

- 用像素（Pixels）表示图像信息
- 像素的集合成为位图（Bitmap）

## Vector Graphics (矢量图形)

- 用几何图形描述（可理解为某种绘图指令）
- 矢量对象可以被描述对象的改变形状、颜色，而位图不行（只能改变某些像素的值）

## 二者的类比

> **前者可以类比为画布上画儿；后者类比为可以在背景上自由拉伸的线和形状**
